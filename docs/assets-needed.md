4. Show character map, relationship graph, timeline, location map, and concept cards.

## 60-second demo video script

2. Show four pain points: names, relationships, places, spatial logic.
3. Show StoryVista extracting characters, scenes, objects, and timeline.
4. Show an interactive atlas with click-to-detail panels.
5. Show actor preparation and worldbuilding examples.
6. End with install command and example prompt.
