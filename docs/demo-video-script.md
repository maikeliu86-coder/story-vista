# Demo Video Script

Target length: 45-60 seconds.

1. Show the single build command and generated output folder.
2. Open the Overview and identify characters, places, events, and relationships.
3. Search for a character and open the evidence drawer.
4. Switch to Timeline, then Actor Mode.
5. Select two characters to show objectives, subtext, actions, and physicality.
6. Open Visual Bible and explain that images are manifest-bound and replaceable.
7. End on `verification-report.md` with zero warnings.

Voice-over should describe what the viewer can verify, not claim generalized literary intelligence beyond the demo.
