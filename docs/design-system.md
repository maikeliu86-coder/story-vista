# Design System

StoryVista uses a quiet Game Codex interface optimized for repeated reading rather than a marketing landing page.

## Layout

- Persistent section navigation
- Dense, scannable codex cards
- Main views for characters, relationships, locations, maps, objects/lore, technology/magic, timeline, gallery, evidence, and settings
- Right Reader Sync Panel: resizable and collapsible on desktop, full-screen drawer on mobile
- Evidence detail drawer for visual profiles and source quotes

## Themes

Theme presets are data-driven through `theme-profile.json`. Current deterministic presets are `literary-archive`, `ancient-chinese-literary`, and `futuristic-sci-fi`. Themes use five-color palettes and spoiler-free atmosphere prompts.

## Highlights

Characters use amber, locations cyan, objects/lore violet, and organizations green. Color is reinforced by underline and hover behavior. UI labels come from locale files, and layouts allow translated strings to wrap.
