## Important

GitHub Topics do not become active just because they appear in the README or in this document. You need to add them manually in the repository web UI:

1. Open the repository on GitHub.
2. Find the `About` panel on the right side.
3. Click the settings icon or `Edit repository details`.
4. Add the topics there and save.
