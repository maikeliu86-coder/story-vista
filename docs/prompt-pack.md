# Prompt Pack

`prompt-pack.md` is the provider-neutral handoff. Each asset includes its ID, entity, asset type, provider recommendation, aspect ratio, style mode, evidence summary, generation notes, expected filename, prompt, and negative prompt.

Provider-specific translations live under `prompts/`. The default style mode is `creative-balanced`: confirmed story facts remain stable while costume, lighting, and composition may be completed plausibly. `evidence-strict` minimizes inference; `cinematic-free` allows broader visual interpretation.
