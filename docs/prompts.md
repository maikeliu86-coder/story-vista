
```text
Use $story-vista to create a 3D spatial map for the planets, ships, routes, rooms, or cities in this story.
```

```text
Use $story-vista to build a responsive interactive atlas that works on desktop, tablet, and mobile. Make character cards clickable, group the character graph by faction or story function, and keep relationship labels readable.
```

```text
Use $story-vista to create a true 3D spatial map: planets should rotate, moons should orbit, ships should move or drift, and model bodies as well as labels should be clickable. Do not use 2D photo cards or flat image stickers.
```

```text

```
