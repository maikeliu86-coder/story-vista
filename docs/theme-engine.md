# Theme Engine

The theme engine identifies broad atmosphere and visual motifs without inferring plot revelations. It outputs `theme-profile.json` with theme ID, confidence, palette, motifs, a spoiler-free background prompt, and background enabled state.

Current deterministic presets:

- `ancient-chinese-literary`: bamboo forest, ink wash, xuan paper, mountain-water landscape
- `futuristic-sci-fi`: deep space, starship, holographic interface, cool light
- `literary-archive`: archival paper, quiet library, annotated map, restrained cinematic light

Background prompts must contain no characters, plot events, text, or later-chapter clues. Settings can disable atmospheric backgrounds.
