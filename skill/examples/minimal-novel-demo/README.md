# Minimal Novel Demo: 吴越夜雨

This fictional Chinese historical-drama excerpt demonstrates the deterministic v0.2 pipeline with five characters, four locations, five relationships, four events, and Actor Mode notes.

```bash
python scripts/storyvista.py build skill/examples/minimal-novel-demo/input.txt --out output/minimal-novel-demo
```

The `expected/` folder contains a checked-in reference build. The source is original project demo text and is not based on a copyrighted novel.
