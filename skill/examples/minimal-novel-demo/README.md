# Minimal Novel Demo

This demo shows the required StoryVista asset workflow:

1. Input a short story text.
2. Extract three characters: Mara Vale, Brother Calen, and Lio.
3. Extract two locations: The Lantern Causeway and Tidemark Archive.
4. Generate `visual-asset-plan.json`.
5. Generate `image-manifest.json`.
6. Give every character a complete portrait prompt.
7. Give every key location a complete location prompt.
8. Use semantic placeholder SVG filenames, not initials-only avatars.

No image provider is assumed. The manifest is ready for a user to replace placeholder files with generated or user-provided images later.
